Seal an artifact

Local tool — runs entirely in your browser; files never leave your machine, and it is not linked from the public site. Open it via a local server (it uses JS modules, so file:// won't work): run python3 -m http.server in the repo root, then visit http://localhost:8000/tools/encrypt.html. Encryption matches slides/viewer.html exactly, so anything sealed here opens in the viewer (and vice-versa with tools/seal.py).
