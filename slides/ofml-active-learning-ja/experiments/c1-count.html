主張 1 · 合成は転移を助ける

図 3c — count 識別と反復外挿（C^k M O）

主張 1 の核心のうち「操作の反復回数（count）を外挿できるか」を切り出した実験。source は各操作を高々 1 回しか含まない 16 本の count probe（\(x\in[-1,1]^2\)）で学習し、target は同一操作 C を \(k=1,\dots,4\) 回反復した \(C^k M O\)（\(x\in[0,1]^2\)、count は真に訓練分布外）で評価する。加法アフィン構造を陽に持つ OFML-linear は、訓練で高々 1 回しか見ていない C の増分を \(k\) 回加法合成して反復を外挿でき（macro NRMSE 0.049）、合成則を陽に持たない count 記述子ベースの Shared-Ens は訓練 count の外で自由に外挿して \(k\) とともに発散する（macro 1.175）。これは「加法アフィンは full-rank probe で全域一意に決まる」という 命題 4(iii)(iv) の直接系である。

実験目的（主張の要約）

「操作の反復回数（count）に関する識別・外挿」を切り出して主張 1 を検証する。source が各操作を高々 1 回しか含まなくても、加法アフィン合成なら target の反復 \(C^k M O\)（\(k=1..4\)）を外挿できることを示す（OFML-linear、macro NRMSE 0.049）。一方、count を陽の合成則なしに扱う Shared-Ens は \(k\) の増加とともに発散し（macro 1.175）、識別可能性の境界を画定する。これは「加法アフィンは full-rank probe から全域一意に決まる」という 命題 4(iii)(iv) の実データ検証にあたる。

実験設定

合成場ベンチマーク上に 2 つの操作族を置く：線形族（各操作が \(2\times2\) アフィン変換の _lin_truth、加法アフィン構造）と非線形族（アフィンでない滑らかな操作 _nl_truth）。各族で、source は 16 本の count probe（各操作の出現回数を系統的に変える。ただしいずれの操作も高々 1 回＝反復は含まない、\(x\in[-1,1]^2\)）を per_proto=32 点ずつノイズ有 \(\sigma=0.02\) で学習し、target は同一操作 C を反復した \(C^k M O\)（\(k=1,2,3,4\)＝C M O / C C M O / C C C M O / C C C C M O、\(x\in[0,1]^2\)）を n_test=80 点のノイズ無し真値で評価する。source と target の入力域を \([-1,1]^2\to[0,1]^2\) とずらし、かつ count \(k\ge2\) を訓練で見せないことで、count は真に分布外（外挿）になる。3 代理モデル（OFML-linear／OFML-Ens／Shared-Ens）を同一データで比較し、12 bench seedを主統計単位に平均する。

※ 生の per-\(k\) 曲線は線形族で集計（図 3c・表 5）。非線形族はスケール横断の要約として macro NRMSE のみ報告する（表 5b）。すべての誤差は各 \(k\)・各族の test 上のスケールで正規化した NRMSEで、生 RMSE は用いない。OFML-linear は最小二乗＋アフィン推定で本来 seed 非依存だが、観測ノイズ実現のばらつきを込めて 12 seed 平均を取る。

指標はすべてマクロ／per-\(k\) NRMSE。各 \(k\) の target について test 上の自スケール \(s_k\) で正規化し、\[ \NRMSE(k)=\sqrt{\mathrm{mean}_{x}\big(\mu_k(x)-f_k(x)\big)^2}\;\big/\;s_k,\qquad s_k=\sqrt{\mathrm{mean}_{x}\big(f_k(x)-\bar f_k\big)^2},\qquad \overline{\NRMSE}=\frac1K\sum_{k=1}^{K}\NRMSE(k),\ \ K=4. \] \(\mu_k\) はモデル予測、\(f_k\) は target \(C^k M O\) の真値、\(\bar f_k\) は test 上平均。\(s_k\) は各 \(k\) の target を自スケールに正規化するので、count が増えて振幅が変わっても誤差を公平に比較できる。「count 盲」とは、Shared-Ens が count を単なる入力特徴として扱い加法合成則を課さないため、訓練で見た count の外では予測が自由に外挿してしまうことを指す。

詳細な設定（seed 分類・手続き・QC・限界）
seed 分類. bench_seed（12 本）が主統計単位で、真のベンチマークモジュール（線形族／非線形族の操作行列・読み出し）を引き直す。派生ストリームを分離：task=bench_seed、pool=bench_seed×7+1、noise=bench_seed×7+2、model=bench_seed。モデル間比較は同一 bench_seed で対応化（ペア化）。

手続き. 各族で 16 本の count probe（各操作 ≤1 回）から source を学習し、\(k=1..4\) の \(C^k M O\) を評価。OFML-linear は操作ごとのアフィン増分を最小二乗で推定し、\(C^k\) を単一 C 増分の \(k\) 回合成として解析的に構成（学習は決定論的、seed はノイズ実現のみ）。OFML-Ens／Shared-Ens は各族の source 全体で 1 度学習し、全 \(k\) の target をゼロショット評価（target evidence なし＝順方向転移のみ）。

QC. test は \(x\in[0,1]^2\) で source 域 \([-1,1]^2\) の内側だが count \(k\ge2\) は訓練外＝真の外挿。test はノイズ無し（train-on-test なし）。Shared-Ens の count 記述子は source と同一実装で、異順序でも byte 単位同一（順序盲＝図 3b で別途露出）。

限界. per-\(k\) の生曲線は線形族のみ（図 3c）。非線形族は macro 要約のみ（表 5b）。OFML-linear の外挿性は加法アフィン仮定に依存し、非線形族では厳密外挿にならない。NRMSE の正規化スケール \(s_k\) は 80 点 test 上の RMS 近似。
実験結果

線形族（図 3c・表 5）。 OFML-linear は全 \(k\) で最良で、count が増えても誤差はほぼ床（NRMSE 0.020→0.082、macro 0.049）＝高々 1 回の操作を反復外挿できている。OFML-Ens は緩やかに劣化（0.177→0.574、macro 0.286）。Shared-Ens は加法合成則を持たず、訓練 count 範囲（\(k\le1\)）を超えると自由外挿して発散（0.190→1.983、macro 1.175）。各 \(k\) 列の最小はすべて OFML-linear。

考察 — 主張をどう支えるか

この実験は主張 1 の「合成は転移を助ける」を、count（反復）の外挿という最も鋭い形で切り出す。source が C を高々 1 回しか含まないにもかかわらず、OFML-linear は単一 C の増分を \(k\) 回加法合成するだけで \(C^k M O\) を全 \(k\) で復元できる（macro 0.049）。これは加法アフィン構造が full-rank probe から全域一意に決定される（命題 4(iii)(iv)）ことの直接系である。対照的に Shared-Ens は count を陽の合成則なしに扱うため、訓練 count の外で予測が拘束を失い、\(k\) とともに発散する — 合成則の有無が外挿可能性の境界を画定する。非線形族でも合成モデルが優位（0.391 vs 0.608）で、加法性が近似的にしか成り立たない場合でも合成の恩恵が残ることを示す。

順方向転移の他の断面は 図 3a（線形 s0 パリティ）・図 3b（order：順序の識別）を、識別可能性の階数監査は Φ/Ψ 監査を、コスト指標での位置づけは 図 2を参照。count（本図）と order（図 3b）は「合成の 2 軸」を成し、両者が揃って初めて terminal-only の識別限界（主張 1）が明確になる。

※ 限界：per-\(k\) の生曲線は線形族のみで、非線形族は macro 要約に留める。OFML-linear の厳密外挿は加法アフィン仮定に依存し、真に非線形な反復には及ばない（表 5b が上限を示す）。Shared-Ens の発散は count 特徴の外挿域挙動に依存し、正則化やアーキテクチャで水準は変わりうるが、合成則を持たない限り \(k\) 依存の劣化は残る。NRMSE の正規化は各 \(k\) の 80 点 test 上のスケール。

理論との対応

本図は 命題 4 の (iii)(iv) を数値化したものである：加法アフィン族では、各操作の増分が full-rank probe（source の count basis）から全域一意に決まり、その和として反復 \(C^k\) が任意の \(k\) に外挿される（\(r^{\mathrm{id}}=0\iff\ker\Phi\subseteq\ker\Psi\)、条件数 \(K\) が有界）。OFML-linear の床張り付き（表 5）はこの一意性の実現であり、Shared-Ens の発散は合成則を持たない仮説クラスが同じ probe から count を同定できないことに対応する。反復が識別可能でない設定（非加法・probe 不足）では、target 終端の能動観測が必要になる（定理 3・命題 5）。
