主張 1 · 合成は転移を助ける（識別可能性の監査）

Φ/Ψ 識別可能性監査 — いつ合成が転移するか

理論の rank 条件を実データで直接測る監査。合成アフィン線形族の真のパラメータ \(\theta\)（23 次元：5 個の \(2\times2\) 操作行列＋読み出し）で、source 行に積んだパラメータ Jacobian を \(\Phi\)、target \((p_\star,q)\) の感度を \(\Psi\) とし、目標関連識別残差 \(r^{\mathrm{id}}=\norm{\Psi(I-P_{\mathrm{row}}(\Phi))}_F/\norm{\Psi}_F\) を計算する。\(r^{\mathrm{id}}=0\iff \ker\Phi\subseteq\ker\Psi\iff\) target が source から識別可能、で、条件数 \(K=\sigma_{\max}(\Psi)/\sigma_{\min}^{+}(\Phi)\)。これは主定理の 命題 4 の rank 条件・条件数を、そのまま数値化したものである。

実験目的（主張の要約）

「識別可能な操作の再結合 \(\Rightarrow\) 順方向転移が成功する」という主張 1 の条件を、真の \(\theta\) における \(\Phi/\Psi\) の階数関係で直接検証する。\(r^{\mathrm{id}}\) が 0 かどうか（＝ \(\ker\Phi\subseteq\ker\Psi\) か）で、その target が source だけから識別できるかが決まり、これが 図 1a の順方向転移 NRMSE をそのまま予言することを示す。決定論的（seed なし）な理論–実験ブリッジ。

実験設定

合成アフィン線形族 _lin_truth（非可換操作 A/B/C/D/M ＋線形読み出し、パラメータ 23 次元）。各シナリオ（source/target の組）で、真の \(\theta\) におけるパラメータ Jacobian を source 行にわたって積んで \(\Phi\)、target 終端 \((p_\star,q)\) にわたって積んで \(\Psi\) とする。指標は \(r^{\mathrm{id}}=\norm{\Psi(I-P_{\mathrm{row}}(\Phi))}_F/\norm{\Psi}_F\)（\(P_{\mathrm{row}}(\Phi)\) は \(\Phi\) の行空間への射影）と \(K=\sigma_{\max}(\Psi)/\sigma_{\min}^{+}(\Phi)\)（\(\sigma_{\min}^{+}\) は \((\ker\Phi)^\perp\) 上の最小特異値）。さらに操作型ごとの source 階数 \(\{A,B,C,D,M\}\) を記録し、どの操作方向が source で励起されていないかを見る。純粋に線形代数的な計算で、乱数・学習を含まない。

※ この監査は 命題 4（\(r^{\mathrm{id}}=0\iff\ker\Phi\subseteq\ker\Psi\iff\row(\Psi)\subseteq\row(\Phi)\)、\(K\le\norm{\Psi}_2/\sigma_{\min}^{+}(\Phi)\)）の直接実装。旧「識別可能性↔容量トレードオフ」図は、識別性を階数で厳密に測るこの監査に置き換えた。

実験結果

\(r^{\mathrm{id}}\) は順方向転移 NRMSE を予言する。 完全に識別可能な s0（\(r^{\mathrm{id}}=0\)）は OFML-linear が近完全に再結合（NRMSE ≈0.04）。source に存在しない操作 D を使う s1 は D 方向が \(\ker\Phi\setminus\ker\Psi\) に入り \(r^{\mathrm{id}}=0.414\)（操作 D の source 階数 = 0）。source probe を減らした s4 は B・C が励起不足になり階数が 15→11 に落ち、残差 \(r^{\mathrm{id}}=0.167\) は小さいものの条件数 \(K=178.5\) が誤差を増幅する。

考察 — 主張をどう支えるか

この監査は主張 1 の適用範囲（転移が成功する条件と失敗する条件の境界）を階数で厳密に画定する。\(r^{\mathrm{id}}=0\)（識別可能）なら合成は転移し（主張 1、s0）、\(r^{\mathrm{id}}>0\)なら terminal-only ではゼロショットで消せない（主張 1、s1）。失敗はprobe 設計の核から生じる：s1 は操作 D が全 source に不在（\(\ker\Phi\setminus\ker\Psi\)）、s7 の三体相互作用も単色・二色 probe の核から同様に \(r^{\mathrm{id}}>0\) を生む。s4 は残差が小さくても \(K\) が大きく、識別が逼迫する領域で余剰容量が同定を不安定化するという直感を定量化する。

理論との対応

本監査は 命題 4 の忠実な実装である：\(r^{\mathrm{id}}=0\iff\ker\Phi\subseteq\ker\Psi\iff\row(\Psi)\subseteq\row(\Phi)\)（命題 4(i)）、条件数 \(K=\sigma_{\max}(\Psi)/\sigma_{\min}^{+}(\Phi)\le\norm{\Psi}_2/\sigma_{\min}^{+}(\Phi)\)（命題 4(ii)）。識別が破れる（\(r^{\mathrm{id}}>0\)）ケースでも、target 終端を直接能動観測すれば終端応答一致性を回復できる（定理 3・命題 5）。
