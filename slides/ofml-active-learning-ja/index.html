Object-Flow Machine Learning:
Operation-Level Surrogate Modeling for Laboratory Workflows

実験プロトコルを、単一の識別子や平坦な特徴ベクトルではなく、実験対象・操作列・測定器からなる構造化入力として扱う。同じ操作型に対応する予測成分をプロトコル間で共有することで、少数の終端観測から未知プロトコルへゼロショット転移し、能動学習で閉ループ補正する。完全な機構シミュレータと単一ブラックボックス代理モデルの中間に位置する、プロトコル構造を保存した代理モデリングである。任意の相互参照はホバーでプレビュー、クリックで該当箇所へ。

3 段階：source で学び、未知プロトコルへ再結合し、target で補正する

同じ操作型の予測成分（同色）を複数プロトコルで共有する。これがプロトコル間知識転移の唯一の機構である。source protocols で共有モジュールを学習し（①）、それらを未知の target protocol へ組み替えてゼロショット予測し（②）、target 上で終端観測を追加しながら能動学習で閉ループ補正する（③）。

実世界の対象はプロトコルである。その抽象は型付き有向非巡回グラフ（DAG）上の共有モジュールの合成であり、各操作型に学習可能な予測成分 \(g_{\theta_j}\) を割り当てる。学習信号は終端測定器ノードの出力だけ — 中間トレースは学習にも評価にも使わない。→ 手法の詳細
  • 点クエリではなくプロトコル全体を扱う。 実験設定 \(x\) と終端測定値 \(y\) を、1 つの構造化された応答として学習する。
  • 操作単位で知識を再利用する。 同じ操作型が異なるプロトコル・位置・出現回数で現れても、同一の予測成分を共有する。
  • 未知プロトコルへゼロショット転移する。 識別可能性が満たされれば、source の観測誤差が target の誤差を制御する（知識転移上界）。
  • 能動学習で閉ループ補正する。 予測不確実性から次の実験 \((p,x)\) を選び、終端観測を追加して残差を補正する。
  • 自律実験の代理モデル基盤。 完全な機構モデルと単一ブラックボックスの中間に位置する構造化代理モデリング。

実験の主張

実験は 2 つの主張に分解される

継続的な surrogate 性能を、target ラベルの累積取得コスト \(b\) に対するマクロ NRMSE 曲線で測る。\(b=0\) が順方向転移、\(b>0\) が能動適応。各カードをクリックすると、その主張を支える実験ページ（実験目的・実験設定・実験結果・考察）へ移動する。すべての実験は terminal-only（各プロトコル実行につき単一の終端スカラー \(y\) のみを観測）で、中間観測アブレーション（主張 1 の 1-C）でのみ学習中に観測器を段階導入する。

① 共通事後モーメント収束（Proposition）

Banach 値 Lévy 上方収束により事後平均・二次モーメントが一様収束し \(\mu_n\to\mu_\infty\)、\(v_n\to v_\infty\)（\(C(\mathcal{X})\) で）。ガウス過程更新に依存しない。

② 残差不確実性の上マルチンゲール（Lemma）

\(H_n=\int v_n\,d\lambda\) は非負上マルチンゲールで、選択点の一段階期待低減量が獲得関数に一致 \(D_n=\Delta^{\mathrm{IVR}}_n(X_{n+1})\)、ゆえに \(\sup_x\Delta^{\mathrm{IVR}}_n(x)\to0\)。

③ 雑音付き IVR/SUR 一致性（主定理）

線形予測子下界＋全台測度の小球質量で \(\left\lVert v_n\right\rVert_\infty\to0\)、ゆえに \(G=\mu_\infty\)、\(\left\lVert\mu_n-G\right\rVert_\infty\to0\)（固定雑音でも潜在関数分散が全域で消失）。

④ OFML target active correction（Theorem）

固定 target protocol を pushforward で \(C(\mathbb{X})\) 値化し、Lipschitz bridge で仮定を discharge。転移識別が破れても target 終端の直接能動観測で終端応答一致性を回復。
