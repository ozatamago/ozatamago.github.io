Theoretical statement and proofs

\(C(\X)\) 値ベイズ能動学習の一致性と OFML target active correction への特化

本ページは理論部を全面的に再構成した版である。旧 OFML 転移上界は主定理から外し、一般の \(C(\X)\) 値ベイズモデルに対する能動学習一致性を一次貢献として据える。主定理は、正しく特定された \(C(\X)\) 値事後に対し、雑音付き \(\IVR/\SUR\) が潜在関数分散 \(v_n\) を全域で消失させ、事後平均 \(\mu_n\) を真の終端関数 \(G\) へ一様収束させることを示す。その後、OFML の固定 target protocol を \(C(\X)\) 値ランダム関数の pushforward として構成し、一般定理を target active correction に特化する。相互参照はホバーでプレビュー、クリックで移動。証明は各結果の「証明を表示」から開ける。

全体アーキテクチャと主張の境界

2 段構成：一般定理 → OFML instance

論理順序は、まず一般の \(C(\X)\) 値ベイズ能動学習定理を full proof で証明し、その後で OFML の target protocol に pushforward で特化する。旧転移上界は独立の大域定理へは持ち上げない。転移側に残すのは、転移が成立しうる十分条件（C1–C5）、線形 rank 命題、および失敗モードの診断だけである。重要なのは転移診断と能動補正の分離である：C1–C5 は source→target の zero-shot 転移を診断するが、target active correction は target 終端を直接観測するため、module 同定に失敗しても、終端応答が \(C(\X)\) 値モデルとして正しく扱える限り一般定理を適用できる。

証明の骨格（雑音付き主定理）

最も非自明なのは \(\sup_x\DeltaIVR_n(x)\to0\ \Rightarrow\ \norm{v_n}_\infty\to0\) である。\(v_n\) がある点で正に残ると、事後二乗平均同程度連続性により近傍でも共分散 \(K_n(u,x)\) が正に残り、全台測度がその近傍に一様な質量を与えるため一段階期待低減量が正に残る — これは \(\sup_x\DeltaIVR_n\to0\) と矛盾する。

\[ \begin{gathered} \mu_n\to\mu_\infty,\quad v_n\to v_\infty \ \text{ in } C(\X) \ \Downarrow\ H_n=\textstyle\int v_n\,d\lambda \ \text{は非負上マルチンゲール} \ \Downarrow\ \sup_x\DeltaIVR_n(x)\to0 \ \Downarrow\ \norm{v_n}_\infty\to0 \ (\text{線形予測子下界}+\text{全台性}) \ \Downarrow\ G=\mu_\infty,\quad \norm{\mu_n-G}_\infty\to0. \end{gathered} \]

1. 一般の \(C(\X)\) 値ベイズモデル

未知の終端関数を \(G:\Omega\to C(\X)\) として扱う。\(\X\) はコンパクト距離空間、事後はガウス過程に限定しない。事後平均 \(\mu_n\) が真の関数 \(G\) へ一様収束すること \(\norm{\mu_n-G}_\infty\to0\) を示すのが目標である。

Assumption1入力空間と潜在関数

\((\Omega,\F,\Pp)\) を確率空間、\((\X,d)\) を非空コンパクト距離空間とする。\(C(\X)\) は一様ノルム \(\norm{g}_\infty=\sup_{x}\abs{g(x)}\) の下で可分バナッハ空間であり、そのボレル空間は標準ボレル空間である。未知の潜在関数 \(G:\Omega\to C(\X)\) はボレル可測で \(\E\norm{G}_\infty^2<\infty\) を満たす。履歴 \((\Hh_n)_{n\ge0}\) は増大する \(\sigma\) 加法族列、\(\Hh_\infty=\sigma(\bigcup_n\Hh_n)\)。

Assumption2正しい事後分布

各 \(n\in\N\cup\{\infty\}\) について、\(\Pi_n=\Law(G\mid\Hh_n)\) を \(G\) の \(\Hh_n\) に関する正則条件付き分布の一つの版とする。

事後モーメント

事後平均・生二次モーメント・共分散核・潜在分散を次で定義する。

\[ \mu_n=\E[G\mid\Hh_n],\quad M_n=\E[G\tensor G\mid\Hh_n],\quad K_n(x,u)=M_n(x,u)-\mu_n(x)\mu_n(u),\quad v_n(x)=K_n(x,x). \]

※ 雑音付き設定で消すべきは観測予測分散（\(=v_n(x)+\)雑音分散、固定雑音下でゼロへ行かない）ではなく、潜在関数分散 \(v_n(x)=\Var(G(x)\mid\Hh_n)\) である。残差不確実性を \(H_n=\int_\X v_n(u)\,\lambda(du)\) と置く。

Lemma1点評価と事後分散の同定

仮定 1・仮定 2 の下で、\(\mu_n(x)=\E[G(x)\mid\Hh_n]\)、\(M_n(x,u)=\E[G(x)G(u)\mid\Hh_n]\) が成り立つ版を取れる。したがって \(v_n(x)=\Var(G(x)\mid\Hh_n)\)、\(K_n(x,u)=\Cov(G(x),G(u)\mid\Hh_n)\)。

証明を表示

点評価 \(\delta_x(g)=g(x)\) は連続線形写像。ボホナー条件付き期待の連続線形写像に対する可換性から \(\delta_x(\mu_n)=\E[G(x)\mid\Hh_n]\)。同様に \((x,u)\) 点評価を \(C(\X\times\X)\) に適用して \(M_n(x,u)=\E[G(x)G(u)\mid\Hh_n]\)。中心化で \(v_n,K_n\) が点ごとの条件付き分散・共分散と一致する。\(\qquad\blacksquare\)

Lemma2閉じた Banach 値 Lévy 上方収束

\(E\) を可分バナッハ空間、\(Z\in L^1(\Omega;E)\)、\(Z_n=\E[Z\mid\F_n]\)、\(Z_\infty=\E[Z\mid\F_\infty]\) をボホナー条件付き期待とする。このとき \(\norm{Z_n-Z_\infty}_E\to0\) ほとんど確実に、かつ \(\E\norm{Z_n-Z_\infty}_E\to0\)。

証明を表示

塔則で \(Z_n=\E[Z_\infty\mid\F_n]\)。\(E\) 可分より、任意の \(\delta>0\) に \(\F_N\) 可測有限値単関数 \(S\) が存在して \(\E\norm{Z_\infty-S}_E<\delta\)。\(n\ge N\) で条件付き Jensen より \(\E\norm{Z_n-Z_\infty}_E\le2\E\norm{Z_\infty-S}_E<2\delta\)（\(L^1\) 収束）。概収束は \(\E\norm{Z_\infty-S_k}_E\le2^{-3k}\) と選び、\(\norm{Z_n-S_k}_E\le\E[\norm{Z_\infty-S_k}_E\mid\F_n]\) に Doob の弱型最大不等式（\(\Pp(\sup_{n\ge N_k}\norm{Z_n-S_k}_E>2^{-k})\le2^{-2k}\)）と Borel–Cantelli を適用して得る。\(\qquad\blacksquare\)

Proposition1共通事後モーメントの一様収束

仮定 1・仮定 2 の下で、ほとんど確実に

\[ \mu_n\to\mu_\infty=\E[G\mid\Hh_\infty]\ \text{ in } C(\X),\qquad M_n\to M_\infty\ \text{ in } C(\X\times\X), \]

したがって \(K_n\to K_\infty\)、\(v_n\to v_\infty\) が \(C(\X)\) で成り立つ。

証明を表示

\(G\in L^1(\Omega;C(\X))\) に 補題 2 を \(E=C(\X)\) で適用して \(\mu_n\to\mu_\infty\)。\(Q(g)(x,u)=g(x)g(u)\) は \(\norm{Q(g)}=\norm{g}_\infty^2\) を満たすので \(G\tensor G\in L^1(\Omega;C(\X\times\X))\)、同補題で \(M_n\to M_\infty\)。\(K_n=M_n-\mu_n\tensor\mu_n\) の一様収束と対角制限で \(v_n\to v_\infty\)。\(\qquad\blacksquare\)

2. 導入例：無雑音 \(\MaxVar\)

無雑音観測を仮定し、\(\MaxVar\) を証明構造の導入として扱う。主目的は「共通事後モーメント収束から分散消失を閉じる」構造を明示することである。

Assumption3無雑音観測・\(\MaxVar\) 近似最大化

各時刻 \(n\) で \(X_{n+1}\) は \(\Hh_n\) 可測、観測は \(Y_{n+1}=G(X_{n+1})\)、\(\Hh_{n+1}=\sigma(\Hh_n,X_{n+1},Y_{n+1})\)。さらに非負誤差 \(\eta_n^{\MaxVar}\ge0\) が存在して \(v_n(X_{n+1})\ge\sup_x v_n(x)-\eta_n^{\MaxVar}\)、\(\eta_n^{\MaxVar}\to0\) ほとんど確実に。

Theorem1無雑音 \(\MaxVar\) 関数学習一致性

仮定 1・仮定 2・仮定 3 の下で、ほとんど確実に \(\norm{v_n}_\infty\to0\)、\(G=\mu_\infty\) in \(C(\X)\)、\(\norm{\mu_n-G}_\infty\to0\)。

証明を表示

無雑音では \(G(X_{n+1})=Y_{n+1}\) が \(\Hh_{n+1}\) 可測なので \(v_{n+1}(X_{n+1})=0\)。命題 1 より \(\norm{v_n-v_{n+1}}_\infty\to0\)、よって \(v_n(X_{n+1})\to0\)。近似最大化から \(\norm{v_n}_\infty\le v_n(X_{n+1})+\eta_n^{\MaxVar}\to0\)、ゆえに \(v_\infty\equiv0\)。可算稠密集合上で \(G(q_j)=\mu_\infty(q_j)\)、連続性で \(G=\mu_\infty\)。最後に \(\norm{\mu_n-G}_\infty=\norm{\mu_n-\mu_\infty}_\infty\to0\)。\(\qquad\blacksquare\)

導入例としての位置づけ

※ \(\MaxVar\) の証明は無雑音補間性 \(v_{n+1}(X_{n+1})=0\) に依存し、簡潔だが固定雑音下では使えない。雑音付き主定理では残差不確実性 \(H_n\) の上マルチンゲール構造へ切り替える。

3. 主定理：雑音付き \(\IVR/\SUR\)

Assumption4加法的・条件付き不偏・有限条件付き分散観測

\(X_{n+1}\) は \(\Hh_n\) 可測、観測は \(Y_{n+1}=G(X_{n+1})+\eps_{n+1}\)。雑音は条件付き不偏 \(\E[\eps_{n+1}\mid G,\Hh_n,X_{n+1}]=0\)、既知定数 \(0\le\tau^2<\infty\) に対し \(\Var(\eps_{n+1}\mid G,\Hh_n,X_{n+1})\le\tau^2\)。

※ 証明で使うのは条件付き不偏性と有限分散上界だけで、劣ガウス性は仮定しない（\(\tau^2=0\) は無雑音を含む）。劣ガウス性は有限標本の集中や実験報告への接続のための任意の強化条件。

Assumption5台が全体である評価測度

\(\lambda\) は \(\X\) 上の有限ボレル測度で \(\supp(\lambda)=\X\)。

Assumption6事後二乗平均同程度連続性

確率 \(1\) の事象上で、ある非減少関数 \(\rho\)（\(\rho(r)\to0,\ r\downarrow0\)）が存在し、全ての \(n\) と \(x,u\in\X\) に対して \(\E[(G(u)-G(x))^2\mid\Hh_n]\le\rho(d(u,x))^2\)。

仮想観測核と雑音付き \(\IVR\) 獲得関数

各 \(n\) について、正則事後核 \(\Pi_n\)、実際の雑音の正則条件付き分布の可測な版である仮想雑音核 \(R_n(\omega,x,g,de)\)、仮想観測 \(Y_x=g(x)+e\) の予測核 \(Q_n^x\)、および仮想事後核 \(\Pi_n^{x,y}\) を可測に固定する（可測性は上の仮定群と標準ボレル性から従う）。仮想事後分散 \(v_n^{x,y}(u)\)、仮想残差不確実性 \(H_{n+1}^{x,y}=\int v_n^{x,y}\,d\lambda\) に対し、候補点 \(x\) の獲得関数を

\[ \DeltaIVR_n(x)=H_n-\int_\R H_{n+1}^{x,y}\,Q_n^x(dy) \]

で定義する。近似最大化ルール：\(x\mapsto\DeltaIVR_n(x)\) は上半連続な版を持ち、\(\Hh_n\) 可測な問い合わせ点 \(X_{n+1}\) が \(\DeltaIVR_n(X_{n+1})\ge\sup_x\DeltaIVR_n(x)-\eta_n^{\IVR}\)、\(\eta_n^{\IVR}\to0\) を満たす（可測選択は \(\X\) のコンパクト性と上半連続性による）。

Lemma3残差不確実性の上マルチンゲール性と獲得関数との一致

\(H_n=\int_\X v_n\,d\lambda\) は非負可積分上マルチンゲール \(\E[H_{n+1}\mid\Hh_n]\le H_n\)。実際に選ばれた点で \(D_n:=H_n-\E[H_{n+1}\mid\Hh_n]=\DeltaIVR_n(X_{n+1})\) であり、\(\sum_n\E[D_n]\le\E[H_0]<\infty\)。したがって近似最大化の下で \(\sup_x\DeltaIVR_n(x)\to0\) ほとんど確実に。

証明を表示

全分散公式より \(v_n(u)=\E[v_{n+1}(u)\mid\Hh_n]+\Var(\E[G(u)\mid\Hh_{n+1}]\mid\Hh_n)\)、ゆえに \(\E[v_{n+1}(u)\mid\Hh_n]\le v_n(u)\)。条件付き Tonelli で \(\E[H_{n+1}\mid\Hh_n]\le H_n\)。仮想核は実際の更新核と一致するので \(\E[H_{n+1}\mid\Hh_n]=\int H_{n+1}^{X_{n+1},y}Q_n^{X_{n+1}}(dy)\)、よって \(D_n=\DeltaIVR_n(X_{n+1})\)。非負上マルチンゲールの望遠和 \(\sum_{k\le N}\E[D_k]=\E[H_0]-\E[H_{N+1}]\le\E[H_0]\) から \(D_n\to0\) a.s.。\(\DeltaIVR_n\ge0\) と近似最大化で \(0\le\sup_x\DeltaIVR_n(x)\le\DeltaIVR_n(X_{n+1})+\eta_n^{\IVR}\to0\)。\(\qquad\blacksquare\)

Lemma4雑音付き \(\IVR\) の線形予測子下界

任意の \(x\in\X\) に対し \(\displaystyle \DeltaIVR_n(x)\ge\int_\X\frac{K_n(u,x)^2}{v_n(x)+\tau^2}\,\lambda(du)\)（\(v_n(x)+\tau^2=0\) の場合は被積分関数を \(0\) と読む）。

証明を表示

\(Y_x=G(x)+\eps_x\)。条件付き不偏性から \(\Covn(G(u),Y_x)=K_n(u,x)\)、\(\Varn(Y_x)\le v_n(x)+\tau^2\)。全分散公式で \(\DeltaIVR_n(x)=\int\Varn(\E[G(u)\mid\Hh_n,Y_x])\,\lambda(du)\)。\(\E[\,\cdot\mid\Hh_n,Y_x]\) は \(L^2\) の \(\sigma(Y_x)\) 上への直交射影なので \(\Varn(\E[G(u)\mid\Hh_n,Y_x])\ge\Covn(G(u),Y_x)^2/\Varn(Y_x)\ge K_n(u,x)^2/(v_n(x)+\tau^2)\)。積分して結論。\(\qquad\blacksquare\)

補助評価（近傍下界と小球質量）

共分散の近傍下界：仮定 6 と条件付き Cauchy–Schwarz から \(\abs{K_n(u,x)-v_n(x)}\le\rho(d(u,x))\sqrt{\norm{v_n}_\infty}\)。一様小球質量：\(\lambda\) が全台でコンパクトなら、任意の \(r>0\) に \(m(r)=\inf_x\lambda(B(x,r))>0\)（有限被覆による）。

Proposition2雑音付き \(\IVR/\SUR\) による全域分散消失

仮定 1–仮定 6（および可測な仮想核・近似最大化）の下で \(\norm{v_n}_\infty\to0\) ほとんど確実に。

証明を表示

背理法。ある \(\delta>0\) と部分列 \((n_j)\)、最大点 \(x_{n_j}\) で \(v_{n_j}(x_{n_j})=\norm{v_{n_j}}_\infty\ge\delta\)。\(V_\star=\sup_n\norm{v_n}_\infty<\infty\)。近傍下界から \(\rho(r)\sqrt{V_\star}\le\delta/2\) となる小さい \(r\) を取れば \(d(u,x_{n_j})補題 4 より

\[ \DeltaIVR_{n_j}(x_{n_j})\ge\int_{B(x_{n_j},r)}\frac{K_{n_j}(u,x_{n_j})^2}{v_{n_j}(x_{n_j})+\tau^2}\lambda(du)\ge\frac{\delta^2}{4(V_\star+\tau^2)}\,m(r)>0 \]

が無限回成り立ち、補題 3 の \(\sup_x\DeltaIVR_n(x)\to0\) と矛盾する。\(\qquad\blacksquare\)

Theorem2雑音付き \(\IVR/\SUR\) による関数学習一致性（主定理）

仮定 1–仮定 6 の下で、ほとんど確実に

\[ G=\mu_\infty\ \text{ in } C(\X),\qquad \norm{\mu_n-G}_\infty\to0. \]
証明を表示

命題 2 で \(\norm{v_n}_\infty\to0\)、命題 1 で \(v_\infty\equiv0\)。可算稠密集合上で \(\Var(G(q_j)\mid\Hh_\infty)=0\) すなわち \(G(q_j)=\mu_\infty(q_j)\) a.s.、連続性で \(G=\mu_\infty\)。\(\mu_n\to\mu_\infty\) 一様より \(\norm{\mu_n-G}_\infty\to0\)。\(\qquad\blacksquare\)

4. モデル水準の十分条件

仮定 6 は事後水準の条件に見えるが、モデルの標本路連続率から検証できる。識別性を結論に近い仮定として置くのではなく、標本路正則性に落とすのが役割である。

Assumption7モデル水準の二乗可積分連続率

ある非減少関数 \(\omega\)（\(\omega(r)\to0\)）と非負確率変数 \(\Lambda\)（\(\E[\Lambda^2]<\infty\)）が存在し、確率 \(1\) で任意の \(x,u\) に対し \(\abs{G(u)-G(x)}\le\Lambda\,\omega(d(u,x))\)。

Proposition3モデル水準連続率 → 事後二乗平均同程度連続性

仮定 7 の下で 仮定 6 が成り立つ。

証明を表示

\((G(u)-G(x))^2\le\Lambda^2\omega(d(u,x))^2\) の条件付き期待で \(\E[(G(u)-G(x))^2\mid\Hh_n]\le\omega(d(u,x))^2\,\E[\Lambda^2\mid\Hh_n]\)。\(M_n^\Lambda=\E[\Lambda^2\mid\Hh_n]\) は閉じた非負スカラーマルチンゲールで a.s. 収束し \(C_\Lambda=\sup_n M_n^\Lambda<\infty\) a.s.。\(\rho(r)=\sqrt{C_\Lambda}\,\omega(r)\) で結論。\(\qquad\blacksquare\)

単一 GP・GPN・BNN での検証方針

※ 単一 GP はカーネル連続率と標本路正則性から。GP 関数ネットワークは各ノードの標本路連続率と合成のリプシッツ安定性を積み上げる。BNN は活性化のリプシッツ性・重み事前の二次モーメント・入力領域のコンパクト性から確率的リプシッツ定数 \(\Lambda\) の二乗可積分性を確認する。

5. 近似推論と獲得関数誤差

分離すべき診断量

主定理は正しい事後と真の獲得関数の近似最大化誤差 \(\eta_n^{\IVR}\to0\) を仮定する。実装（MCMC, SMC, VI, Laplace, 深層アンサンブル）では次を分離診断する。

\[ a_n=\norm{\widehat\mu_n-\mu_n}_\infty,\quad b_n=\norm{\widehat v_n-v_n}_\infty,\quad d_{\Delta,n}=\norm{\widehat\DeltaIVR_n-\DeltaIVR_n}_\infty,\quad \eta_n^{\mathrm{opt}}=\sup_x\widehat\DeltaIVR_n(x)-\widehat\DeltaIVR_n(\widehat X_{n+1}). \]

\(d_{\Delta,n}\to0\) かつ \(\eta_n^{\mathrm{opt}}\to0\) なら \(\eta_n^{\IVR}\le2d_{\Delta,n}+\eta_n^{\mathrm{opt}}\to0\) で制御できる。固定近似族の VI・固定 Laplace・固定サイズ ensemble は一般に \(a_n,b_n,d_{\Delta,n}\to0\) を保証しない。理論条件と整合するのは成長予算 MCMC/SMC、容量を増やす近似族、または関数空間上で一貫性を検証できる近似推論に限られる。

6. OFML target active correction への特化

一般 \(C(\X)\) 定理を OFML に特化する。target は固定 protocol \(p_\star\) と単一終端観測器 \(q\in\Obs(p_\star)\)。転移側は診断条件に限定し、収束は target 終端の直接能動観測から得る。

転移診断（残す十分条件）と削除する主張

source protocols だけから target 応答を保証する旧形式の大域転移上界は主定理として採用しない。source 側経験誤差 \(\dtrain(\theta,\theta')=\sup_{p\in\Psrc}\sup_{x}\norm{\Eval(p;\theta;x)-\Eval(p;\theta';x)}\)、target 終端誤差 \(d_{\star,q}(\theta,\theta')=\sup_x\norm{\Eval_q(p_\star;\theta;x)-\Eval_q(p_\star;\theta';x)}\) に対し、次の条件は \(d_{\star,q}\) を source information から診断するためのものである。

Assumption8OFML 転移診断の十分条件 C1–C5（任意 C6）

C1（vocabulary）target の全 node type は source と同じ module vocabulary に含まれる。C2（semantics）同じ type 名は同じ操作的意味を持つ。C3（coverage）target で type \(j\) が受け取る局所入力 \(\Ucal_j^\star\) は source probe の \(\Ucal_j^{\mathrm{src}}\) で残差 \(\rho_j\ge0\) を残して覆われる。C4（target-relevant identifiability）射影 \(\Pi_{j,q}\)・定数 \(K_j<\infty\)・識別残差 \(r_{j,q}^{\mathrm{id}}\ge0\) が存在し \(\norm{\Pi_{j,q}F_j^\theta(z)-\Pi_{j,q}F_j^{\theta'}(z)}\le K_j\dtrain(\theta,\theta')+r_{j,q}^{\mathrm{id}}+\rho_j\)。C5（Lipschitz stability）各 module は局所入力について Lipschitz で局所摂動が終端へ有限に伝播。C6（任意 realizability）参照 target 応答が class 内で表現可能（well-specified 版でのみ）。

条件役割破れたときの典型的失敗
C1 vocabularytarget 操作が既知 module として存在する未知操作は source 誤差から制御できない
C2 semantics同じ type が同じ状態更新を意味する同名操作の意味ずれで source 学習が target に移らない
C3 coveragetarget 局所入力が source probe の近傍にある外挿残差 \(\rho_j\) が残る
C4 identifiabilitytarget に効く方向が source 観測で見える\(\ker\Phi\setminus\ker\Psi\) 方向が残り \(r^{\mathrm{id}}>0\)
C5 stability局所誤差が有限倍率で終端へ伝播深い合成や高感度観測で証明定数が崩れる
C6 realizability真の target 応答が class 内にあるwell-specified でなく近似誤差付きの主張になる
診断としての知識転移上界（旧主定理を降格）

C4 の per-module 不等式を DAG の下流 Lipschitz 増幅 \(\Lambda_v(p_\star,q)\) に沿って足し合わせると、形式的には

\[ d_{\star,q}(H,F^\dagger)\ \le\ C_{\mathrm{KT}}(p_\star,q)\,\dtrain(H,F^\dagger)+R_{\mathrm{id/cov}}(p_\star,q) \]

という知識転移上界が得られる（\(C_{\mathrm{KT}}=\sum_v\Lambda_v K_{\tau(v)}\)、\(R_{\mathrm{id/cov}}=\sum_v\Lambda_v(r^{\mathrm{id}}_{\tau(v),q}+\rho_{\tau(v)})\)）。本稿ではこれを独立の転移定理へは持ち上げず、あくまで zero-shot 転移が期待できるかの診断として扱う。識別残差 \(r^{\mathrm{id}}>0\) や被覆残差 \(\rho>0\) が残る場合の一致性は、次の target active correction に委ねる。

線形特徴づけ

加法アフィンモデルでは \(\Eval(p;\theta;x)=\varphi_{p,x}^\top\theta\)。source probe の感度作用素を \(\Phi\)、target 終端の感度作用素を \(\Psi\) とする。

Proposition4加法アフィン OFML の識別可能性（rank 条件）

(i) \(r^{\mathrm{id}}=0\ \Longleftrightarrow\ \ker\Phi\subseteq\ker\Psi\ \Longleftrightarrow\ \row(\Psi)\subseteq\row(\Phi)\)。(ii) このとき C4 最小定数は \(K=\sup_{\delta\theta\notin\ker\Phi}\frac{\norm{\Psi\delta\theta}}{\norm{\Phi\delta\theta}}\le\frac{\norm{\Psi}_2}{\sigma_{\min}^{+}(\Phi)}\)。(iii) full-rank probe で係数が一意なら対応方向で被覆残差 \(\rho=0\)。(iv) \(\Phi\) 列フルランク・経験誤差 \(\le\delta\) なら \(\norm{\widehat\theta-\theta^\dagger}\le\delta/\sigma_{\min}(\Phi)\)。

証明を表示

(i) 有限な支配定数 \(K\) の必要条件が \(\ker\Phi\subseteq\ker\Psi\)。逆に成り立てば \(\Psi\) は商空間 \(\ThetaSpace/\ker\Phi\) 上の有界作用素を誘導し \(K<\infty\)。線形代数で \(\row\) 条件と同値。(ii) \((\ker\Phi)^\perp\) 上への射影で \(\norm{\Phi\delta\theta}\ge\sigma_{\min}^{+}(\Phi)\norm{\delta\theta}\)、\(\norm{\Psi\delta\theta}\le\norm{\Psi}_2\norm{\delta\theta}\)。(iii) full-rank probe で係数一意なら target 値も同じ係数から決まる。(iv) 標準的最小二乗摂動評価。\(\qquad\blacksquare\)

※ 実験対応：rank 条件（線形 order の識別）は 図 4、count 外挿は 図 5、識別可能なレジームの存在証明は 図 1a（主張 1 の 1-A）。三体相互作用による \(r^{\mathrm{id}}>0\) の失敗は 図 1b（s7）、識別可能性の rank 監査は Φ/Ψ 監査（主張 1 の 1-B）。

失敗モード：三体相互作用と過剰潜在次元

三体相互作用：色水混合で三体項 \(\gamma\,rgb\) を許すと単色・二色 probe で常に \(rgb=0\) なので \(\gamma\in\ker\Phi\)。三色 target で \(rgb\ne0\) なら \(\gamma\notin\ker\Psi\)。よって \(\ker\Phi\subseteq\ker\Psi\) が破れ \(r^{\mathrm{id}}>0\)（probe 設計の核から生じる、非線形性の有無とは無関係）。過剰潜在次元：gauge が source と target 両方で観測不能なら \(\ker\Phi\cap\ker\Psi\) に入り無害。source で観測不能だが target に効く方向は \(\ker\Phi\setminus\ker\Psi\) に入り \(r^{\mathrm{id}}>0\)。過剰容量は容量そのものでなく target-relevant identifiability の問題として整理される。

pushforward と Lipschitz bridge

Lemma5OFML target posterior の \(C(\Xtar)\) pushforward

\(\Xtar=\X_{p_\star}\) を compact metric とする。(a) 各 \(\theta\) で \(x\mapsto\Eval_q(p_\star;\theta;x)\) が連続、(b) \((\theta,x)\mapsto\Eval_q(p_\star;\theta;x)\) が可測、(c) envelope \(B(\theta)\) が \(\sup_x\abs{\Eval_q(p_\star;\theta;x)}\le B(\theta)\)、\(\E[B(\theta)^2]<\infty\) を満たすとする。このとき \(T_{p_\star,q}(\theta)(x)=\Eval_q(p_\star;\theta;x)\) は可測で \(G=T_{p_\star,q}(\theta)\) は \(C(\Xtar)\) 値二乗可積分確率変数。さらに \(\Pi_n^G=(T_{p_\star,q})_\#\Pi_n^\theta=\Law(G\mid\Hh_n^{\mathrm{tar}})\)。

証明を表示

可算稠密集合上の点評価で \(C(\Xtar)\) のボレル \(\sigma\) 加法族が生成され、(b) で各点可測なので \(T_{p_\star,q}\) は可測。(c) で \(\E\norm{G}_\infty^2\le\E[B(\theta)^2]<\infty\)。任意のボレル \(A\) に \((T_{p_\star,q})_\#\Pi_n^\theta(A)=\Pp(G\in A\mid\Hh_n^{\mathrm{tar}})\) より pushforward は正しい事後。\(\qquad\blacksquare\)

Lemma6OFML Lipschitz bridge

各 node \(v\) の局所入力 injection \(\iota_v\) が design input について Lipschitz（定数 \(c_v\)）、各 module \(F_{\tau(v)}^\theta\) が局所入力について Lipschitz（定数 \(L_{\tau(v)}(\theta)\)）とする。\(\Lambda_{p_\star,q}(\theta)=\sum_{v\preceq q}c_v\sum_{\pi:v\to q}\prod_{w\in\pi}L_{\tau(w)}(\theta)\) が有限で \(\E[\Lambda_{p_\star,q}(\theta)^2]<\infty\) なら、\(G_\theta(x)=\Eval_q(p_\star;\theta;x)\) は \(\abs{G_\theta(u)-G_\theta(x)}\le\Lambda_{p_\star,q}(\theta)\,d(u,x)\) を満たす。したがって 仮定 7（\(\omega(r)=r\)）が成り立ち、命題 3 により 仮定 6 が従う。

証明を表示

state 差 \(e_v=\norm{s_v^\theta(u)-s_v^\theta(x)}\) を DAG の位相順序で帰納。親なしで \(e_v\le L_{\tau(v)}c_v d(u,x)\)、親ありで \(e_v\le L_{\tau(v)}(c_v d(u,x)+\sum_{a\in\Pa(v)}e_a)\)。展開すると各 node の design input 由来差が \(v\to q\) の各 path 上の Lipschitz 定数積で増幅され \(e_q\le\Lambda_{p_\star,q}(\theta)d(u,x)\)。二乗可積分性で \(\E[\Lambda^2]<\infty\)。\(\qquad\blacksquare\)

有界版と独立版

※ spectral norm 制約・有界活性化・weight clipping で \(L_j(\theta)\le\bar L_j\) を置けば \(\Lambda_{p_\star,q}\) は決定論的に有界。独立な random Lipschitz constants では \(\E[\prod_j L_j(\theta)^2]<\infty\) を直接確認する。

target active correction の一致性

Theorem3OFML target active correction の終端応答一致性

固定 target protocol \(p_\star\)、単一終端 \(q\)、compact \(\Xtar\) を考える。(i) 補題 5 の条件で pushforward posterior が正しい、(ii) target 観測は 仮定 4、(iii) \(\lambda\) は \(\Xtar\) 上全台（仮定 5）、(iv) 可測な仮想核、(v) \(\IVR/\SUR\) 近似最大化、(vi) 補題 6 の条件、を仮定する。このとき \(\norm{v_n}_\infty\to0\)、\(\norm{\mu_n-G}_\infty\to0\) a.s.。特に well-specified \(f_{p_\star}^{\dagger}=G\) なら \(\norm{\mu_n-f_{p_\star}^{\dagger}}_\infty\to0\) a.s.

証明を表示

補題 5 で 仮定 1・仮定 2 を、(ii)–(v) で 仮定 4・仮定 5 と可測核・近似最大化を、補題 6＋命題 3 で 仮定 6 を discharge。主定理（定理 2）を \(\X=\Xtar\) に適用して結論。\(\qquad\blacksquare\)

※ 実験対応：コスト軸で統一した順方向転移＋ target 能動適応は 図 2（コスト指標曲線）、全代理での AL レースは S4（主張 2）。

Proposition5転移識別が破れても target active correction で救える

命題 4 の意味で \(r^{\mathrm{id}}>0\) が残り source だけからの zero-shot 転移が target 残差を持つとする。それでも 定理 3 の仮定が満たされ target 終端 \(f_{p_\star}^{\dagger}\) が well-specified なら \(\norm{\mu_n-f_{p_\star}^{\dagger}}_\infty\to0\) a.s.

証明を表示

\(r^{\mathrm{id}}>0\) は source protocols だけからは target-relevant 方向を識別できないことを意味するが、定理 3 は source-only transfer を仮定しない。Source data \(\Dsrc\) は初期履歴 \(\Hh_0^{\mathrm{tar}}\) として posterior を初期化するだけで、その後 target 終端観測が \(\lambda\) 全台の \(\IVR/\SUR\) rule で追加される。適用して結論。\(\qquad\blacksquare\)

深さ限界と中間観測の動機づけ

chain depth \(D\) の target で bridge 係数は典型的に \(\Lambda_{p_\star,q}(\theta)\asymp\prod_{k=1}^{D}L_k(\theta)\) を含む。\(\E[\prod_k L_k(\theta)^2]\) が深さとともに増大すると 仮定 6 の \(\rho\) 係数は緩くなる（反例ではなく、有限予算での分散崩壊を遅く見せうる証明定数上の問題）。また有限粒子 Monte Carlo で \(d_{\Delta,n}\not\to0\) なら近似最大化条件を満たせず、定理 3 は実装方策には適用できない。深い非ガウス pushforward では標本共分散・仮想観測 posterior の推定が不安定なため、中間観測や内部 node の部分評価は \(\Lambda_{p_\star,q}\) と \(d_{\Delta,n}\) の両方を改善する設計動機を持つ（＝中間観測＝主張 1 の 1-C の理論的裏付け）。

7. 限界と実験の主張への接続

本理論は実験の 2 主張と次のように対応する。主張 1（転移・b=0）：線形操作かつ容量一致で rank 条件 \(\row(\Psi)\subseteq\row(\Phi)\) が成り立てば \(r^{\mathrm{id}}=0\) となり、合成が順方向転移を助ける（命題 4）。三体相互作用のように probe 設計の核から \(r^{\mathrm{id}}>0\) が残ると terminal-only ではゼロショットで消せず、深さ限界を緩める中間観測が \(\Lambda_{p_\star,q}\) と \(d_{\Delta,n}\) を改善して条件を回復する（命題 6）。以上が 主張 1。主張 2（能動学習・b>0）：識別が破れても、target 終端の直接能動観測（全台 \(\lambda\) + 正しい事後）により 定理 3・命題 5 で終端応答一致性を回復し、構造の優位で誤差を早く下げる（主張 2）。
限界

※ (1) 消えるのは観測予測分散でなく潜在関数分散 \(v_n\)。(2) 主定理は加法的・条件付き不偏・有限分散観測に依存し、一般観測核へは局所識別性（Fisher 情報量型下界）が要る。(3) 雑音付き \(\MaxVar\) は本稿の主対象でない。(4) 近似推論の一貫性は自動には得られず、固定近似族は事後分散を過小・過大評価しうる。Misspecified で best-in-class limit への収束を主張するには擬真値・posterior concentration・misspecified BvM 型の別仮定が必要。
